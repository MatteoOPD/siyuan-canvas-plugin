# SiYuan Canvas MVP

## User Story

Als SiYuan-Nutzer:in möchte ich Dokumente, einzelne Blöcke und freie Gedanken auf einer Fläche anordnen und mit beschrifteten Pfeilen verbinden, damit ich Zusammenhänge ohne zweite Notizkopie visuell erkunden kann.

## Akzeptanzkriterien

1. Toolbar erstellt/öffnet Canvas; der Graph wird beim Öffnen wiederhergestellt.
2. Fläche: unbegrenzt wirkendes Pan/Zoom, verschiebbare Karten.
3. Dokument- und Blockkarten laden aktuellen Inhalt aus SiYuan.
4. Suche/Drop fügt Referenzen ein; freie Textkarten sind editierbar.
5. Gerichtete, optional beschriftete Pfeile lassen sich erstellen.
6. Der Editor speichert Referenzänderungen über die Kernel-Block-API in die Originalquelle.
7. Graph-JSON wird nur über SiYuan `/api/file/*` gespeichert.

Nicht enthalten: Kollaboration, KI-Layout, Zeichnen, Mobile-Optimierung oder ein Editor auf jeder kleinen Karte.
